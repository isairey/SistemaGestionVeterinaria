# 🐾 Sistema de Gestión para Clínica Veterinaria

Aplicación web desarrollada en **Laravel** para la administración integral de una clínica veterinaria.  
Permite gestionar pacientes (mascotas), dueños, citas, tratamientos y más de forma eficiente.

---

## 🚀 Descripción

Este sistema está diseñado para facilitar el control y organización de una clínica veterinaria, permitiendo:

- Registro de mascotas y dueños 🐶🐱  
- Gestión de citas médicas 📅  
- Historial clínico 🩺  
- Control de tratamientos 💊  
- Administración de usuarios 👨‍⚕️
